# 🌞 Daily Motivation Dashboard

A React app that fetches and displays random motivational quotes with a like/save system.

## Features
- ✅ Fetches random quotes from DummyJSON API
- ✅ Like & unlike quotes
- ✅ Save liked quotes with localStorage persistence
- ✅ Search through saved quotes
- ✅ Loading shimmer animation
- ✅ Fallback quotes if API is unavailable

## Tech Used
- React (useState, useEffect, useCallback)
- DummyJSON Quotes API
- localStorage for persistence
- CSS-in-JS for styling

## How to Run
```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Learning Outcomes
Array state updates, re-fetching data, toggle interactions, API integration
